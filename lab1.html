Инструменты: Для поиска данных будем исполльзовать:

Название Назначение
2ip.ru портал сетевых сервисов
Nmap утилита, предназначеная для сканирования ip-сетей, определение состояния сети (портов и сетевых служб)
REG.ru регистрация доменных имен
Linkonavt.ru определение используемых web-приложений

План выполнения:
С помощью каждого из выше перечисленных сервисов собираем следующие признаки:
1) С помощью сервиса 2ip.ru определим следующие признаки:

  • ip-адрес
  • Местонахождение
  • Телефон
  • e-mail

2) Просканируем порты, используя операционную систему Linux, выполнив команду: 

nmap <домен_компании>/<ip_адрес> -p <номер_порта>


3) При помощи сервиса REG.ru определим кто занимается хостингом данной компании;
4) Используем Linkonavt.ru, чтобы определить web-приложения по домену.
